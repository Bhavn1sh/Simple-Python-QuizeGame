Welcome to the Simple Python Quiz Game, a fun and interactive way to test your Python knowledge! This game contains 5 multiple-choice questions about various topics in Python, and you can earn points based on the correctness of your answers.

How to Play
Each question is worth 2 points.
The game will ask you 5 questions related to Python programming.
You need to type your answer, and the game will immediately check if it's correct.
After all questions have been answered, your total score will be displayed.
Game Features
Simple Python syntax: The code is written in basic Python to ensure it's easy to understand and modify.
Score Tracking: The game tracks your score as you answer each question.
Interactive User Input: The game takes input from the user and provides instant feedback on whether the answer is correct or incorrect.
Final Score: After completing all questions, your final score is displayed, along with a thank you message for playing.
